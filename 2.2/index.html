Sans-serif

Sans-serif-fonter används ofta för rubriker då utseendet är mer simpelt och avskalat. Exempel på sans-serif-font som rubrik nedan.

Rubrik

Monospace

Monospace-fonter är fonter där alla bokstäver i fonten har samma bredd. Det används bland annat för att visa upp exempelkod på en sida. Exempel på monospace-font visas nedan.

public class HelloWorld {

 public static void main(String []args) {
  System.out.println("Hello World");
 }
}

Enheter

Nyckelord

CSS har sju nyckelord som justerar storleken i förhållande till den angivna basstorleken. Dessa är:

  • xx-small
  • x-small
  • small
  • medium
  • large
  • x-large
  • xx-large

%

Procent kan användas för att göra procentuella storleksförändringar i förhållande till basstorleken. Procenten beräknas efter eventuella arv.

em

Em fungerar på exakt samma sätt som procent och 1em är samma sak som 100%. Till exempel blir då 2em 200% och 0.75 em 75%.

Färg

color

color-egenskapen färg på text. Se exempel.

CSS

.color {
 color: red;
}

HTML

<p class="color">Fugiat consectetur ut culpa eu veniam enim id aute pariatur. Eu ullamco enim eiusmod duis mollit aliqua eiusmod consectetur culpa laboris magna. Dolore nulla cupidatat commodo dolor est eiusmod aliqua magna esse do nisi deserunt nulla.</p>

Teckensnitt

font

font-egenskapen är en sammansättning av flera font-egenskaper (font-style, font-variant, font-weight, font-size/line-height, font-family) som gör det möjligt att ändra utseende på text. Se exempel.

CSS

.font {
 font: italic bold small-caps 18px 'Courier New', Courier, monospace;
}

HTML

<p class="font">Custom font text</p>

Resultat:

Custom font text

Listor

list-style

list-style-egenskapen är en sammansättning av flera list-egenskaper (list-style-type, list-style-position, list-style-image). Se exempel nedan.

CSS

.list-style {
 list-style: square inside;
}

HTML

<ul class="list-style">
 <li>item 1</li>
 <li>item 2</li>
 <li>item 3</li>
</ul>

Resultat:

  • item 1
  • item 2
  • item 3
