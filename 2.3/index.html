2.3 Marginaler, kantlinjer och paddning

Boxmodell

Det finns vissa generella parametrar som används för att hantera utfyllnad och placering av element. Dessa används för att organisera elements förhållande till varandra på en sida.

Innehåll

Längst inne i boxmodellen finns innehållet av elementet. Detta är det som vi ser på en sida såsom text, bilder etc.

Padding

Padding används för att skapa utrymme mellan innehållet och elementets kantlinje och finns direkt runt innehållet.

Kantlinje

Kantlinjen finns direkt utanför padding-utrymmet och kan stylas på olika sätt som visas längre ner i denna guide.

Margin

Margin är utrymmet som finns utanför kantlinjen och fungerar på samma sätt som padding, dvs skapar utrymme till nästkommande eller föregående element.

Marginaler

margin

margin-egenskapen används för att skapa utrymme runt element. Border används också för att förtydliga denna egenskap. Se exempel.

CSS

.p-border {
 border: 2px solid cyan;
}

.p-border-margin {
 border: 2px solid red;
 margin: 50px 30px 50px 30px;
}

Kantlinjer

border

border-egenskapen används specificera vilken stil, bredd och färg en kantlinje kan ska ha. Egenskapen är sammansättning av border-width, border-style och border-color. Se exempel.

CSS

.h4-border {
 border: 4px solid springgreen;
}

.h5-border {
 border: 4px solid darkmagenta;
}

HTML

<h4 class="h4-border">posuere morbi leo urna molestie</h4>
<h5 class="h5-border">tortor at risus viverra adipiscing</h5>

Resultat:

posuere morbi leo urna molestie

tortor at risus viverra adipiscing

border-color

border-color kan ha en till fyra värden och sätter färg på en borders fyra olika väggar. Se exempel.

CSS

.h4-border-color {
 border: 4px solid;
 border-color: red green blue yellow;
}

HTML

<h4 class="h4-border-color">neque laoreet suspendisse interdum consectetur</h4>

Resultat:

neque laoreet suspendisse interdum consectetur

border-style

border-style sätter stilen på kantlinjen för vardera sida. Se exempel.

CSS

.h4-border-style {
 border: 4px;
 border-style: dotted solid dashed groove;
}

HTML

<h4 class="h4-border-color">erat imperdiet sed euismod nisi</h4>

erat imperdiet sed euismod nisi

border-width

border-width sätter tjockleken på kantlinjen för vardera sida. Se exempel.

CSS

.h4-border-width {
 border: solid;
 border-width: 4px 2px 10px 6px;
}

HTML

<h4 class="h4-border-width">in aliquam sem fringilla ut</h4>

Resultat:

in aliquam sem fringilla ut

Padding

padding

padding används för att skapa utrymme runt ett visst element men innanför kantlinjen. Detta är alltså inte samma sak som margin som hanterar utrymmet utanför kantlinjen. Se exempel.

CSS

.p-border {
 border: 2px solid cyan;
}

.p-border-padding {
 border: 2px solid red;
 padding: 50px 30px 50px 30px;
}

HTML

<p class="p-border">Tempor eu veniam excepteur est in quis tempor ut commodo incididunt amet incididunt
dolor. Labore ullamco adipisicing ut reprehenderit culpa nulla.</p>
<p class="p-border-padding">Exercitation adipisicing cupidatat deserunt exercitation. Eiusmod Lorem nisiad
ex incididunt fugiat est quis sit fugiat magna do minim enim.</p>
<p class="p-border">Tempor eu veniam excepteur est in quis tempor ut commodo incididunt amet incididunt
dolor. Labore ullamco adipisicing ut reprehenderit culpa nulla.</p>

Outlines

outlines

outline är en kantlinje som dras direkt utanför en eventuell border-kantlinje. outline består av outline-width, outline-style och outline-color. border är även med i exempel för bättre tydlighet. Se exempel nedan.

CSS

.h4-outline {
 border: 2px solid blue;
 outline: 4px dotted red
}

HTML

<h4 class="h4-outline">arcu non odio euismod lacinia</h4>

Resultat:

arcu non odio euismod lacinia

Synlighet

visibility

Egenskapen specificerar om ett element ska vara synligt eller ej. Elementet tar upp plats trots att det inte är synligt. Se exempel.

CSS

.h4-visibility {
 visibility: hidden;
}

HTML

<h4 class="h4-visibility">dignissim suspendisse in est ante</h4>

Resultat:

dignissim suspendisse in est ante

clip-path

clip-path används till visa ett specifikt område av ett element. Se exempel.

CSS

.img-clip-path {
 clip-path: circle(50%);
 height: 30%;
 width: 30%;
}

HTML

display

display definierar vilken typ av visningsbox som ska användas för elementet. Typerna är uppdelade i yttre och inre visningstyper där de yttre hanterar hur visningsboxen ska bete sig i förhållande till element på samma nivå, och de inre hanterar hur barn till elementet ska hanteras. Se exempel.

CSS

.span-display {
 display: block;
 color: blue;
}

Transparant visningsbox

Man kan göra ett element eller en visningsbox ransparant genom att använda sig av rbga-funktionen eller genom egenskapen opacity. rbga-funktionen tar emot värden mellan 0 till 255 för röd, grön och blå samt ett alphavärde mellan 0 och 1 som bestämmer hur transparant färgen ska vara. Opacity definerar hur transparant ett element ska vara genom alpha värde mellan 0 och 1. Se exempel.

CSS

.div-rgba-filled {
 width: 200px;
 height: 100px;
 background-color: rgba(137, 20, 255, 1);
}

.div-rgba-opacity {
 width: 200px;
 height: 100px;
 background-color: rgba(137, 20, 255, 0.2);
}

HTML

<div class="div-rgba-filled"></div>
<div class="div-rgba-opacity"></div>

Resultat:

CSS

.div-boder-radius {
 width: 200px;
 height: 100px;
 border: 4px groove orangered;
 border-radius: 50px 10px 100px 20px;
}

HTML

<div class="div-border-radius"></div>

Resultat:

Multipla bakgrundsbilder

Man kan applicera multipla bakgrundsbilder till element och dessa läggs då som flera lager på varandra. Man använder sig av background-egenskapen och dess olika subegenskaper förutom background-color som inte fungerar vid multipla bakgrunder. Se exempel.

CSS

.div-multiple-backgrounds {
 width: 800px;
 height: 800px;
 background-image: url(https://upload.wikimedia.org/wikipedia/commons/4/47/PNG_transparency_demonstration_1.png), linear-gradient(to bottom, rgba(135, 65, 240, 0.8), rgba(100, 234, 54, 0.8));
 background-repeat: no-repeat, no-repeat;
 background-position: center, center;
}

HTML

<div class="div-multiple-backgrounds"></div>

Resultat:

Skuggor

Man kan använda sig av box-shadow-egenskapen för att ge ett element skugga. Se exempel.

CSS

.div-box-shadow {
 width: 200px;
 height: 100px;
 background-color: rgba(137, 20, 255, 0.2);
 box-shadow: 5px 10px lightslategray;
}

HTML

<div class="div-box-shadow"></div>

Resultat:

Gradienter

Gradienter kan skapas med hjälp av CSS-funktionerna linear-gradient och radial-gradient. Se exempel.

CSS

.div-linear-gradient {
 width: 200px;
 height: 200px;
 background-image: linear-gradient(to right, rgba(123, 200, 140, 1), rgba(255, 142, 124, 1));
}

.div-radial-gradient {
 width: 200px;
 height: 200px;
 background-image: radial-gradient(rgba(123, 200, 140, 1), rgba(255, 142, 124, 1));
}

HTML

<div class="div-linear-gradient"></div>
<div class="div-radial-gradient"></div>
