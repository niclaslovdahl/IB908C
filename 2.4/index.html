2.4 Bilder och bakgrunder

Bakgrund

background

background-egenskapen är en förkortning av background-color, background-image, background position, background-repeat, background-origin, background-clip och background-attatchment. Med hjälp av dessa egenskaper kan man skapa och modifiera bakgrunder. Egenskaperna har delats upp i exempel för ökad tydlighet.

CSS

body {
 background-image: linear-gradient(to bottom, rgba(123, 22, 235, 1), rgba(255, 255, 255, 0));
 background-repeat: no-repeat;
 background-size: auto 200px;
}

h1 {
 background-color: tomato;
 padding: 10px;
 border: 4px solid greenyellow;
 background-clip: content-box;
}

li {
 background-image: linear-gradient(to right, red, green, blue, yellow, orange);
 background-size: 400px auto;
 background-repeat: no-repeat;
 background-position: right;
}

Hantering av flytande text bredvid en bild

Vill man att text ska flyta bredvid en bild i en container kan man använda sig av egenskapen float. Man deklarera var i containern man vill att bilden ska flyta. Vill man göra undantag för flytandet för vissa element kan man använda egenskapen clear för det elementet vilket gör att det elementet visas efter det flytande elementet. Margin används för att justera avståndet mellan bild och text. Se exempel.

CSS

img {
 width: 20%;
 margin: 0px 20px 20px 20px;
 float: left;
}

h3 {
 clear: left;
}
