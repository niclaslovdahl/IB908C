2.5 Navigation

Länktillstånd

Det finns olika länktillstånd som en webbläsare kan identifiera. Dessa består av en länk som har besökts, en länk som inte har besökts, en länk där pekaren svävar över och en länk som blir klickad. Man kan hantera stilen för dessa olika tillstånd genom att addera tillståndet till selektorn. Se exempel.

a:link, a:active, a:visited och a:hover

CSS

a:link {
 color: darkorange;
}

a:active {
 color: blue;
}

a:visited {
 color: green;
}

a:hover {
 color: crimson;
}

HTML

<a href="https://www.google.se/">https://www.google.se/</a>

Resultat:

https://www.google.se/

Övrigt

cursor

Med cursor kan man ändra hur pekaren ser ut när man pekar på ett visst element. Se några exempel nedan.

CSS

.crosshair-cursor {
 cursor: crosshair;
}

.move-cursor {
 cursor: move;
}

.progress-cursor {
 cursor: progress;
}

.zoom-in-cursor {
 cursor: zoom-in;
}

HTML

<h4 class="crosshair-cursor">Crosshair</h4>
<h4 class="move-cursor">Move</h4>
<h4 class="progress-cursor">Progress</h4>
<h4 class="zoom-in-cursor">Zoom-in</h4>

Resultat:

Crosshair

Move

Progress

Zoom-in
