2.6 Tabeller och formulär

Tabeller

table-layout

table-layout bestämmer hur tabellen ska visas. Sätts den till auto så används en automatisk tabell-algoritm som justerar cellernas storlek till cellernas innehåll. Används fixed så kommer tabellens bredd bestämmas av första raden celler eller av width om den har ett värde. Se exempel.

CSS

.auto-table {
 table-layout: auto;
 width: 50px;
 border-collapse: collapse;
}

.auto-table * {
 border: 2px solid black;
}

.fixed-table {
 table-layout: fixed;
 width: 50px;
 border-collapse: collapse;
}

.fixed-table * {
 border: 2px solid black;
}
