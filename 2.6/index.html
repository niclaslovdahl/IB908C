2.6 Tabeller och formulär

Tabeller

table-layout

table-layout bestämmer hur tabellen ska visas. Sätts den till auto så används en automatisk tabell-algoritm som justerar cellernas storlek till cellernas innehåll. Används fixed så kommer tabellens bredd bestämmas av första raden celler eller av width om den har ett värde. Se exempel.

CSS

.auto-table {
 table-layout: auto;
 width: 50px;
 border-collapse: collapse;
}

.auto-table * {
 border: 2px solid black;
}

.fixed-table {
 table-layout: fixed;
 width: 50px;
 border-collapse: collapse;
}

.fixed-table * {
 border: 2px solid black;
}

Formulär

När man ska göra formulär är en bra idé att använda sig av label-taggen. När man ska styla formuläret kan man då få problem då labeltaggen även används för till exempel radioknappar. Man vill då inte att varje label i ett formulär ska hanteras som en egen rad eller egen kategori. För komma runt detta är det en bra idé att använda span-taggen där det finns multipla inputs för en kategori. För att styla formuläret i en fin lista delger man istället de label- och span-taggar man vill ska vara en egen kategori en specifik klass.

Det är även en bra idé att använda display-egenskapen på denna klass satt till inline-block då man då har möjligheten att bestämma width och height på dessa taggar.

När det kommer till att styla specifika input-element i ett formulär är möjligheterna väldigt olika. Det finns möjligheter att ändra färg och form men generellt så vill man försöka hålla ett formulär så konsekvent man kan för en bättre användarupplevelse.
