2.7 Sidlayout

Fixerad layout

position, bottom, top, left och right

position används för att bestämma hur ett element ska placeras ut på en sida. bottom, top, left och right bestämmer vart på sidan elementet ska placeras. Denna div är placerad med position: absolute samt bottom: 0px; relativt till sin förälder.

float

Egenskapen float används för att specificera hur ett element ska "flyta" i förhållande till andra element i ett dokument. På detta sätt kan man få element att visas upp bredvid varandra genom att använda float: left; eller float: right;. Utan float kommer element på sidan visas upp vertikalt efter varandra.

z-index

Denna egenskapen bestämmer i vilken ordning element som överlappar ska visas. Element visas i ordning lägst värde längst bak och högst värde längst fram. Den gula lådan under denna text har z-index: -1 och placeras därför bakom texten.

clear

Egenskapen bestämmer om element är tillåtna att flyta bredvid ett annat. Denna låda har egenskapen clear: both vilket gör att inga element får flyta på någon sida och elementet får därför en ny rad.
